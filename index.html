How to Play
1You're given a Start Actor and a Target Actor.
2Name a movie the current actor appeared in, then name a co-star from that movie to hop to next.
3Every move is verified live against TMDB's real database — no guessing allowed!
4Reach the Target Actor in as few hops as possible.
5Example: RDJ → Tropic Thunder → Jack Black → The Minecraft Movie → Jason Momoa ✓
Easy — well-known modern actors, short connections.
Medium — character actors and less obvious bridges.
Hard — classic Hollywood stars requiring multiple hops.
